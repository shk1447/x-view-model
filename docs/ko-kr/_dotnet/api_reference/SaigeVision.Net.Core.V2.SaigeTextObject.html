    Struct SaigeTextObject

    (OCR) 텍스트를 검출하는 OCR 검사의 결과가 담긴 구조체입니다. OCR기능은 글자의 영역을 검출하는 부분과 해당 영역의 글자 종류를 인식하는 부분으로 나뉘어있습니다.
    SaigeVision OCR에서는 글자의 영역을 polygon 형태(2차원 점의 배열)로 제공하며 문자열 정보를 같이 담은 구조체로 검사결과를 제공합니다.
