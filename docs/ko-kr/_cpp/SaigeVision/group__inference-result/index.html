    SaigeGetInferenceResultClassActivationMap¶

    (CLS) Class activation map 배열을 받아 옵니다. 

    int SaigeGetInferenceResultClassActivationMap (
    SaigeInferenceResult result,
    unsigned char ** classActivationMap
)

    • Class activation map은 딥러닝 모델이 이미지를 특정 클래스로 분류할 때 이미지의 어떤 영역이 얼마나 영향을 끼쳤는지 나타내는 heatmap입니다.
    • 픽셀 값은 0 ~ 255 범위의 정수로 표시되며 값이 큰 픽셀일수록 분류 결과에 큰 영향을 끼쳤다고 볼 수 있습니다.
    • 형상은 (원본 이미지 높이 H, 원본 이미지 너비 W)입니다.
    • 코드상에서는 싱글 포인터로 표현되는 1차원 배열이며 행 우선 방식을 따릅니다. 즉 i번째 행, j번째 열의 값은 data[W*i + j]입니다.

    Parameters:

    • result 추론 결과가 담긴 결과 핸들입니다.
    • classActivationMap [out] Class activation map 배열입니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    int SaigeGetInferenceResultMask (

    Parameters:

    • result 추론 결과가 담긴 결과 핸들입니다.
    • mask [out] Mask 배열입니다.

    Returns:

    SaigeGetInferenceResultScoremapSEG¶

    (SEG) Scoremap 배열을 받아 옵니다. 

    int SaigeGetInferenceResultScoremapSEG (
    SaigeInferenceResult result,
    unsigned char ** scoremap
)

    • SEG의 scoremap으로 IAD의 scoremap과는 다릅니다.
    • 형상은 (OK를 포함하는 클래스 개수, 원본 이미지 높이 H, 원본 이미지 너비 W)입니다.
    • 이미지의 i번째 행, j번째 열 픽셀이 c번째 클래스로 예측되었다면, (c, i, j) 픽셀에 해당 클래스로 예측한 score 값이 0 ~ 255 범위의 정수로 표시됩니다.
    • 코드상에서는 싱글 포인터로 표현되는 1차원 배열이며 행 우선 방식을 따릅니다. 즉 c번째 클래스, i번째 행, j번째 열의 값은 data[W*H*c + W*i + j]로 얻을 수 있습니다.

    Parameters:

    • result 추론 결과가 담긴 결과 핸들입니다.
    • scoremap [out] Scoremap 배열입니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.
