    SaigeVisionAPI.h¶

    Go to the source code of this file.

    SaigeVision 추론 기능에 필요한 핵심적인 함수, 상수, 구조체, 플래그 등이 정의되어 있습니다.

    Classes¶

    Type Name
    struct SaigeHandleConfig
    추론 핸들 설정 사항이 담긴 구조체입니다. GPU 메모리 및 추론 시간에 영향을 미칠 수 있는 항목들입니다.
    struct SaigeRoiInfo
    ROI 정보가 담긴 구조체입니다. ROI 모드와 그에 대응하는 파라미터, 그리고 공통적으로 사용되는 blind mask 정보로 구성되어 있습니다.
    struct SaigeModelInfo
    모델 정보가 담긴 구조체입니다.
    struct SaigeTextFormatting
    (OCR) 인식하고자 하는 글자의 포맷을 지정하는 옵션입니다. 글자 수에 관계 없이 인식하고자 하는 글자집합을 설정하는 방식과, 글자의 위치별로 인식하고자 하는 글자집합을 설정하는 두 가지 방식이 있습니다.
    struct SaigeInferenceOption
    추론 옵션이 담긴 구조체입니다.
    struct SaigeRotatedBox
    회전 가능한 직사각형의 정보가 담긴 구조체입니다.
    struct SaigeDetectedObject
    Object detection(DET)을 통해 검출된 영역의 정보가 담긴 구조체입니다.
    struct SaigeEllipse
    타원 정보가 담긴 구조체입니다.
    struct SaigeSegmentedObject
    Segmentation(SEG) 또는 Anomaly Detection(IAD)을 통해 검출된 영역의 정보가 담긴 구조체입니다.
    struct SaigeTextObject
    텍스트를 검출하는 OCR 검사의 결과가 담긴 구조체입니다.
    struct SaigeEngineMetadata
    MLOps 기능 반영 시 안내 예정.
    struct SaigeInferenceMetadata
    MLOps 기능 반영 시 안내 예정.
