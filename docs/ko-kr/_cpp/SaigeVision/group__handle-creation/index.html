    Public Functions Documentation¶

    SaigeCreateInferenceHandle¶

    추론 핸들을 생성합니다. 

    int SaigeCreateInferenceHandle (
    SaigeInferenceHandle * handle,
    const int projectType,
    const char * modelFilePath,
    const int deviceIndex,
    SaigeHandleConfig * config,
    const unsigned int flags
)

    원하는 작업에 맞게 프로젝트 타입과 모델 파일 경로를 올바르게 설정해 주시기 바랍니다. 설정한 프로젝트 타입과 모델 파일이 가진 프로젝트 정보가 일치하지 않을 시 에러가 발생합니다.

    Parameters:

    • handle[out] 사용할 추론 핸들입니다.
    • projectType 프로젝트 타입을 나타내는 정수 값입니다.
    • modelFilePath 모델 파일의 경로입니다.
    • deviceIndex 추론에 사용할 장치의 인덱스입니다. GPU를 사용할 경우 GPU의 인덱스(0 이상의 정수), CPU를 사용할 경우 -1을 입력합니다.
    • config 추론 핸들 설정 사항이 담긴 구조체입니다.
    • flags 설정하고자 하는 항목들입니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    See also: SaigeInferenceHandle, SaigeHandleConfig, SaigeDestroyInferenceResult
