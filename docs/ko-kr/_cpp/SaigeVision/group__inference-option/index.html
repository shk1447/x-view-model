    추론 옵션¶

    모델에 담긴 추론 옵션 기본값을 확인하거나 추론 옵션을 새롭게 설정합니다.

    Classes¶

    Type Name
    struct SaigeTextFormatting
    (OCR) 인식하고자 하는 글자의 포맷을 지정하는 옵션입니다. 글자 수에 관계 없이 인식하고자 하는 글자집합을 설정하는 방식과, 글자의 위치별로 인식하고자 하는 글자집합을 설정하는 두 가지 방식이 있습니다.
    struct SaigeInferenceOption
    추론 옵션이 담긴 구조체입니다.

    Public Functions¶

    Type Name
    int SaigeSetInferenceOption (SaigeInferenceHandle handle, SaigeInferenceOption * option, const unsigned int flags)
    추론 옵션값을 추론 핸들에 설정합니다.
    int SaigeGetInferenceOption (SaigeInferenceHandle handle, SaigeInferenceOption * option)
    추론 핸들로부터 추론 옵션값을 받아 옵니다.

    Public Functions Documentation¶

    SaigeSetInferenceOption¶

    추론 옵션값을 추론 핸들에 설정합니다. 

    int SaigeSetInferenceOption (
    SaigeInferenceHandle handle,
    SaigeInferenceOption * option,
    const unsigned int flags
)

    Parameters:

    • handle 사용할 추론 핸들입니다.
    • option 설정하고자 하는 옵션값이 담긴 구조체입니다.
    • flags 새롭게 설정하고자 하는 옵션 항목들입니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    See also: SaigeInferenceOption

    SaigeGetInferenceOption¶

    추론 핸들로부터 추론 옵션값을 받아 옵니다. 

    int SaigeGetInferenceOption (
    SaigeInferenceHandle handle,
    SaigeInferenceOption * option
)

    Parameters:

    • handle 옵션값을 가져올 추론 핸들입니다.
    • option [out] 옵션값을 담을 구조체입니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    See also: SaigeInferenceOption
