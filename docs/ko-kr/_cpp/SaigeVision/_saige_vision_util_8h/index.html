    SaigeDrawImage¶

    이미지를 디스플레이합니다. 

    int SaigeDrawImage (
    const void * image,
    const int width,
    const int height,
    const int channels,
    const double alpha,
    const int type,
    const bool invert,
    const int waitMS
)

    Parameters:

    • image 디스플레이할 이미지 데이터입니다.
    • width 이미지의 너비입니다.
    • height 이미지의 높이입니다.
    • channels 이미지의 채널 수입니다.
    • alpha 이미지 데이터의 픽셀 값에 곱해지는 수입니다.
    • type 이미지 데이터 타입입니다.
    • invert true로 설정하면 색상이 반전되어 표시됩니다.
    • waitMS 디스플레이 시간입니다. 단위는 밀리초이며 0으로 설정할 시 디스플레이 창을 직접 종료해야 합니다.

    alpha 값 설정하기

    이미지가 unsigned char 데이터일 때는 픽셀 값의 범위가 0~255이 되도록 설정하는 것을 권장합니다.

    이미지가 float 데이터일 때는 픽셀 값의 범위가 0~1이 되도록 설정하는 것을 권장합니다.

    type 설정하기

    이미지가 unsigned char 데이터일 때는 'SV_U8'를 입력합니다.

    이미지가 unsigned short 데이터일 때는 'SV_U16'를 입력합니다.

    이미지가 float 데이터일 때는 'SV_F32'를 입력합니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    See also: SaigeGetInferenceResultScoremapSEG, SaigeGetInferenceResultScoremapIAD, SaigeGetInferenceResultMask

    SaigeDrawImageOverBase¶

    한 이미지 위에 다른 이미지를 겹쳐 디스플레이합니다. 

    int SaigeDrawImageOverBase (
    const void * base_img,
    const void * img,
    const int width,
    const int height,
    const int channels_base,
    const int channels_img,
    const int type_base,
    const int type_img,
    const double alpha,
    const bool invert,
    const int waitMS
)

    Parameters:

    • base_img 배경이 될 이미지 데이터입니다.
    • img 배경 위에 표시할 이미지 데이터입니다.
    • width 두 이미지의 너비입니다. 너비가 같은 두 이미지를 사용해야 합니다.
    • height 두 이미지의 높이입니다. 높이가 같은 두 이미지를 사용해야 합니다.
    • channels_base 배경 이미지의 채널 수입니다.
    • channels_img 배경 위에 표시할 이미지의 채널 수입니다.
    • type_base 배경 이미지의 데이터 타입입니다. 설정 방법은 SaigeDrawImage 를 참고해 주세요.
    • type_img 배경 위에 표시할 이미지의 데이터 타입입니다. 설정 방법은 SaigeDrawImage 를 참고해 주세요.
    • alpha 배경 위에 표시할 이미지의 픽셀 값에 곱해지는 수입니다. 배경 이미지에는 영향이 없습니다. 설정 방법은 SaigeDrawImage 를 참고해 주세요.
    • invert true로 설정하면 배경 위에 표시할 이미지의 색상이 반전되어 표시됩니다. 배경 이미지에는 영향이 없습니다.
    • waitMS 디스플레이 시간입니다. 단위는 밀리초이며 0으로 설정할 시 디스플레이 창을 직접 종료해야 합니다.

    Returns:

    errorCode 정상 작동 시 0을 리턴하고, 에러 발생 시 에러코드를 리턴합니다.

    See also: SaigeGetInferenceResultClassActivationMap
